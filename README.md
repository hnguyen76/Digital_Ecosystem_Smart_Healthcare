# Digital Ecosystem Smart Healthcare Dashboard
https://hnguyen76.github.io/Digital-Ecosystem-Smart-Healthcare/
A healthcare analytics dashboard built from the
`digital_ecosystem_smart_healthcare_dataset.csv` dataset.

Created by Hieu Nguyen.

## What It Shows

- Standard-library Python data cleaning pipeline.
- Cleaned analysis-ready CSV and data quality report.
- Patient risk mix across Critical, High Risk, Moderate Risk, and Stable cohorts.
- Executive KPIs for acuity, alerts, digital readiness, and network latency.
- Interactive cohort and metric filters.
- Clinical benchmark bars, digital readiness scatter plot, priority watchlist, and portfolio insights.

## Clean The Data

```powershell
python scripts/clean_data.py
```

The script creates:

- `data/processed/smart_healthcare_cleaned.csv`
- `data/processed/data_quality_report.json`

## Run Locally

This project is a static dashboard with no package install step.

```powershell
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Dataset

The raw CSV contains 4,000 smart healthcare records with clinical vital signs,
connected-device metrics, care operations fields, and patient health status labels.
The dashboard reads the cleaned CSV generated by `scripts/clean_data.py`.
